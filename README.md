# Sales-Dashboard-Task
#Sales Dashboard Project#

Project Overview

This project focuses on designing an interactive Sales Dashboard using Power BI to help business stakeholders analyze sales performance and make data-driven decisions.

Objective

To create an interactive dashboard that provides insights into sales, profit, orders, and regional performance.

Tools Used

- Power BI
- Microsoft PowerPoint
- GitHub
- CSV Dataset

Dataset

Sample Sales Dataset containing:

- Order ID
- Order Date
- Product Information
- Sales
- Profit
- Region
- Category

Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Total Orders
- Category-wise Sales
- Region-wise Profit

Dashboard Features

- KPI Cards
- Sales Trend Analysis
- Category Analysis
- Regional Analysis
- Interactive Slicers and Filters
- Business Insights Visualization

Files Included

- Sales_Data.csv
- Sales_Dashboard.pbix
- Dashboard_Screenshot.png
- PPT_Summary.pptx
- README.md

Key Insights

- Sales performance can be monitored over time.
- Top-performing categories and regions can be identified.
- Interactive filters help users explore data efficiently.
- Dashboard supports business decision-making.

Outcome

Successfully developed an interactive Sales Dashboard in Power BI and presented business insights through visual analytics.
